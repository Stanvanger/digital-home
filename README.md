## 🚀 Ver en local

Abre `index.html` directamente en tu navegador, o usa un servidor local:

```bash
# Con Python (si lo tienes instalado)
python -m http.server 8080

# Con Node.js / npx
npx serve .
```

Luego visita: `http://localhost:8080`
